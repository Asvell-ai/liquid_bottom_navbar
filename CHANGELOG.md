## 0.1.0

- Initial package release.
- Added `LiquidBottomNavBar` with liquid animation and drag support.
- Added `LiquidNavItem` model for item-level icons and labels.
- Added `LiquidNavStyle` for reusable visual configuration.
- Added optional blur/glass effect.
- Added badge support.
- Added public blob size, stretch, and wobble tuning parameters.
- Added example app and full documentation.
